# 六爻排盘 — 二期需求方案（参考图驱动）

> 基于 `起卦.png`、`时间.png`、`paipan.png` 三张设计参考图分析，结合用户需求整理。

---

## 一、起卦界面重构

### 1.1 七种起卦方式 Tab 切换

参考 `起卦.png`，HomeView 中增加 Tab 栏，支持 7 种方式：

| Tab | 功能 | 当前状态 |
|-----|------|----------|
| 摇钱 | 掷3枚硬币，6次成一卦 | 已有 CoinToss，需改造 |
| 手工 | 逐爻选择阴阳老少 | 已有 ManualInput |
| 自动 | 系统随机生成 | 已有 |
| 时间 | 取当前时间+可选修改 | 已有 DateTimeInfo，需改为独立入口 |
| 数字 | 用户输入数字起卦 | 新建 |
| 汉字 | 输入汉字笔画起卦 | 新建 |
| 卦名 | 选择卦名+变爻 | 已有 DivinationForm |

Tab 栏样式：灰色背景，选中 tab 底部红色指示条，文字灰色/红色切换。

### 1.2 每项起卦方式均包含通用信息区

所有 7 种方式顶部统一显示（参考 `起卦.png`）：

- **问题类型**：8 个 pill 按钮（感情-男/感情-女/事业/财运/健康/出行/考试/其他）
- **具体问题**：文本输入框（placeholder: "具体问题（可选）"）
- **性别**：单选按钮（男/女）
- **起卦时间**：默认当前时间，可修改

### 1.3 六爻从下到上排列

参考 `起卦.png`，爻的排列顺序从下到上：
- 最下方 = 初爻（第一次掷币结果）
- 最上方 = 上爻（第六次掷币结果）
- 每爻左侧标注位置名（初/二/三/四/五/上）

### 1.4 摇币起卦改造

- 每次掷币后，显示 3 枚独立随机的铜钱正反面（每枚正反各 50%）
- 使用 coin SVG 贴图，3 枚横向排列
- 3 枚组合决定爻的阴阳老少（三正=老阳、二正=少阳、一正=少阴、零正=老阴）
- 已有结果的爻列表纵向排列，从下到上

### 1.5 备注与操作

参考 `起卦.png`：
- 排盘预览区域显示在本卦和变卦下方，含纳甲标注
- 备注区含：占卜问题、占卜结果（textarea）、标签（已验证/待验证/错卦/参考/教学）
- 底部两个按钮："重新起卦"（次）和 "查看完整排盘"（主）

## 二、摇币随机逻辑

### 2.1 三个硬币独立随机

当前 `tossCoin()` 使用 `Math.random()` 一次生成 0-3 的整数来决定正反面数量。改为：
- 每枚硬币独立调用 `Math.random() >= 0.5` 判定正反
- 三枚硬币的 SVG 图案根据结果独立显示
- 组合结果决定爻的类型不变

### 2.2 动画

每次掷币时三枚硬币同时翻转（CSS 3D 动画），落地后显示结果

## 三、时间模块重构

### 3.1 神煞全面显示

参考 `时间.png`，DateTimeInfo 组件显示结构：

| 区域 | 内容 | 样式 |
|------|------|------|
| 起卦时间 | 格式化时间 + 节气 + 已过天数 + 区间 | 灰标签+绿值 |
| 四柱 | 八字大红粗体单独一行 | `text-[#8b0000] text-lg` |
| 月建/日辰 | 大字地支+小字干支五行 | 橙色背景卡片 |
| 柱支 | 年/月/日/时四柱干支五行 | 两列 grid |
| 空亡 | 地支+旬 | 红色标注 |
| 神煞 | 所有神煞+对应地支 | 紫色标注 |

### 3.2 神煞数据完备

需确保以下神煞全部存在并显示：

天乙贵人、驿马、桃花（咸池）、劫煞、亡神、天德贵人、月德贵人、文昌贵人、华盖、将星、禄神、羊刃

每条标注对应地支和所属柱位。

### 3.3 排盘界面时间不可修改

进入 ResultView 后，时间仅展示（只读布局），不提供编辑按钮。时间修改仅在起卦阶段进行。

## 四、排盘界面重构

### 4.1 传统排盘表

参考 `paipan.png` 的 grid 布局：

**神煞行**（可开关，位于表格上方）：紫色神煞名 inline 显示

**表头 6 列**（本卦+变卦时）：
| 六神 | 本卦（卦名） | 动爻 | 世应 | 变卦（卦名） | 六神 |

**每行数据**（从上爻 = 第6行 到 初爻 = 第1行）：
- 六神列：青龙/朱雀/勾陈/腾蛇/白虎/玄武
- 本卦列：爻线符号 → 六亲 → 旺衰/空亡/暗动标记 → 纳甲干支（五行）
- 动爻列：有动爻显示红色 →，无则空白
- 世应列：红色 ●世 / 蓝色 ○应
- 变卦列（如有）：同本卦结构
- 六神列（变卦的六神，右对齐）

### 4.2 伏神位置

伏神不在每行中 inline 显示，改为在排盘表**下方独立区域**（`bg-amber-50`），列出每个有伏神位置的详细信息（六亲+干支+五行）。

### 4.3 底部信息

卦宫（五行）、变宫（五行）、卦辞（可切换展开）

### 4.4 行样式

- 每行数据水平居中对齐
- 行间使用虚线分割（`border-b border-dashed`）
- 隔行背景色交替
