排盘引擎核心逻辑与京房六爻、渊海子平等传统经典完全一致。所有纳甲、六亲、六神、世应、空亡、三合局、神煞、暗动、伏神、反吟伏吟等规则均验证通过。八字计算使用精确节气算法，年柱以立春为界，月柱以节为界，日柱以子时为界，符合传统命理标准。神煞系统包含 23 种（超过标准 22 种），参数正确。

## 建议
1. 汉字起卦的笔画计算采用的是启发式散列算法（非真实笔画数），对于依赖精确笔画的占卜场景，建议增加真实笔画数查表功能作为选项。
2. 伏神判定逻辑当前判断的是"六亲类型"是否在本卦中出现，而非具体爻位的"缺失"——传统伏神规则在某些流派中以"伏神在本卦同爻位有对应六亲"为补充判定，可根据需要增加流派选项。
