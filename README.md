# 六爻排盘

> 传统六爻纳甲筮法排盘工具 — 在线起卦·排盘·记录·分享

在线使用，无需安装。支持七种起卦方式，完整展示纳甲、六亲、六神、世应、空亡、神煞、三合局、反吟伏吟等全维度信息。

---

## 🚀 在线使用

部署后直接浏览器访问即可，纯前端应用，无需服务器。

```bash
# 本地运行
cd liuyao
npm install
npm run dev
# 浏览器打开 http://localhost:5173/liuyao/
```

生产构建：

```bash
npm run build   # 产物在 dist/ 目录，部署到任意静态服务器
```

详见 [部署指南](docs/deploy-guide.md)。

## ✨ 功能概览

### 七种起卦方式

| 方式 | 用法 |
|------|------|
| **摇钱** | 模拟掷三枚硬币，逐爻投掷六次，每爻显示三正/二正/一正/三反结果 |
| **手工指定** | 直接选择每爻阴阳（少阳/少阴/老阳/老阴），支持回退修改 |
| **自动** | 一键随机生成完整卦象 |
| **时间** | 指定日期时间自动推算卦象 |
| **数字** | 输入三个数字起卦 |
| **汉字** | 输入汉字，按笔画起卦 |
| **卦名** | 从六十四卦库中选择本卦，指定动爻得变卦 |

### 完整排盘信息

- **本卦与变卦** — 卦象可视化对比，动爻高亮
- **四柱八字** — 精确计算年、月、日、时四柱，节气分年柱
- **纳甲装卦** — 按八卦纳干纳支规则装卦
- **六亲排布** — 父母、兄弟、官鬼、妻财、子孙
- **六神安放** — 青龙、朱雀、勾陈、腾蛇、白虎、玄武
- **世应定位** — 按卦在宫中的位置定世应
- **空亡旬空** — 按日柱计算旬空，标注空亡爻位
- **神煞标注** — 天乙贵人、驿马、桃花、劫煞、亡神、禄神、羊刃、文昌、华盖
- **三合局检测** — 申子辰、亥卯未、寅午戌、巳酉丑
- **反吟伏吟** — 本卦与变卦间的反吟伏吟关系
- **伏神飞神** — 以宫卦各爻六亲为伏神
- **旺相休囚死** — 以月建定旺衰

### 信息开关

排盘表支持灵活显隐：卦辞 / 三合局 / 反吟伏吟 / 伏神 / 旺衰 / 神煞 / 空亡

### 历史记录

自动保存最近 100 条记录，支持查看、删除、清空、JSON 导入/导出。

### 导出分享

生成分享链接 / 导出 TXT / 导出 JSON，一键复制排盘文本。

## 📖 使用引导

### 起卦流程

1. 进入首页，在起卦方式标签中选择（默认摇钱）
2. 按所选方式操作：摇钱需逐爻掷币六次，自动则一键生成
3. 起卦完成后自动显示卦象预览和排盘结果
4. 点击「查看完整排盘」进入详情页

### 排盘解读

结果页包含：
- 起卦时间、节气、四柱八字、月建日建
- 本卦与变卦并排展示（世应、纳甲、动爻标注）
- 三合局、反吟伏吟附加信息
- 排盘详情表（每爻的六神、六亲、伏神、干支、五行、旺衰、神煞、空亡）
- 信息开关控制显隐
- 分享/导出按钮

### 卦库查阅

底部导航进入「卦库」，按八宫浏览全部 64 卦，点击查看卦辞、爻辞、象传。支持「以此卦起卦」快捷起卦。
