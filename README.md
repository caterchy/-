# 六爻排盘

> 基于 Vue 3 + TypeScript 的六爻纳甲筮法排盘工具，支持多种起卦方式与完整排盘引擎。

基于 Vue 3 + TypeScript + Vite 构建的六爻纳甲筮法排盘工具。支持手动起卦（掷硬币模拟）和自动起卦，完整展示本卦与变卦、六亲、六神、世应、纳甲、空亡、神煞、三合局、反吟伏吟等纳甲筮法所需的全维度信息。

## 功能特性

- **三种起卦方式**：掷硬币模拟（逐爻投掷）、手工指定（直接选择爻象）、自动随机生成
- **完整四柱八字**：根据起卦时间精确计算年、月、日、时四柱，支持节气划分年柱边界
- **本卦与变卦**：自动展示本卦并依据动爻生成变卦，卦象可视化对比
- **纳甲装卦**：严格按八卦纳干纳支规则装卦，计算每爻的天干地支
- **六亲排布**：以卦宫五行为"我"，生克关系定六亲（父母、兄弟、官鬼、妻财、子孙）
- **六神安放**：按日柱天干起青龙，顺排六神（青龙、朱雀、勾陈、腾蛇、白虎、玄武）
- **世应定位**：按卦在宫中的位置（纯卦、一世至五世、游魂、归魂）定世应
- **空亡计算**：根据日柱定旬空，标注空亡爻位
- **神煞标注**：支持天乙贵人、驿马、桃花、劫煞、亡神、禄神、羊刃、文昌、华盖等九种神煞
- **三合局检测**：自动检测卦中是否存在申子辰、亥卯未、寅午戌、巳酉丑三合局
- **反吟伏吟**：自动检测本卦与变卦间的反吟、伏吟关系
- **伏神计算**：以宫卦（纯卦）各爻六亲为伏神
- **旺相休囚死**：以月建地支判断各爻五行的旺衰状态
- **信息开关**：灵活控制排盘表中卦辞、三合局、反吟伏吟、伏神、旺衰、神煞、空亡等信息的显隐
- **历史记录**：本地存储最多 100 条历史排盘记录，支持查看、删除、清空
- **导出分享**：支持导出为纯文本文件与 JSON 文件，支持生成可分享的 URL 链接
- **响应式设计**：移动端优先，底部导航栏，适配手机与桌面

## 快速开始

### 环境要求

- Node.js >= 18.0
- npm >= 9.0

### 安装

```bash
cd liuyao
npm install
```

### 开发模式

```bash
npm run dev
```

启动后在浏览器打开 http://localhost:5173 即可使用。支持热更新。

### 生产构建

```bash
npm run build
```

构建产物输出至 `dist/` 目录，可直接部署至静态服务器。

### 预览构建产物

```bash
npm run preview
```

## 核心功能模块说明

### 类型系统

`src/types/index.ts` 定义了完整的 TypeScript 类型体系，涵盖：

- **基础枚举**：天干（TianGan）、地支（DiZhi）、五行（WuXing）、八卦（BaGua）、卦宫（GuaGong）、爻类型（YaoType）、六亲（LiuQin）、六神（LiuShen）、神煞（ShenShaType）、旺衰（WangShuai）
- **数据结构**：爻（Yao）、纳甲（Najia）、爻位详情（YaoDetail）、卦详情（GuaDetail）、四柱（Zhu）、八字（BaZi）、三合局（SanHe）、反吟伏吟（FanYinFuYin）、空亡（KongWang）、伏神（FuShen）、神煞标注（ShenShaMark）
- **完整排盘结果**（PaipanResult）：包含唯一标识、起卦时间、八字、本卦、变卦、空亡、三合局、反吟伏吟
- **显示开关**（DisplayOptions）：七个布尔开关控制排盘表中卦辞、三合局、反吟伏吟、伏神、旺衰、神煞、空亡的显隐

### 排盘引擎

排盘引擎是项目的核心计算模块，位于 `src/engine/` 目录，各模块职责清晰、可独立验证。主流程由 `paipan.ts` 编排，按以下顺序生成完整的排盘结果：

1. **起卦**：使用 `coin.ts` 获取六个爻（自动掷币或手动输入），转换为二进制编码（阳 = 1，阴 = 0）
2. **八字**：使用 `bazi.ts` 计算起卦时间的四柱八字，含年柱（立春分界）、月柱（五虎遁）、日柱（基准日期法）、时柱（五鼠遁）
3. **定卦**：通过六位二进制码查询 `hexagrams.ts` 确定卦名、上下卦、卦宫、卦辞及爻辞
4. **世应**：使用 `shiying.ts` 根据卦在宫中的位置（0 纯卦 / 1-5 一世至五世 / 6 游魂 / 7 归魂）定位世爻和应爻
5. **纳甲**：依据 `naja.ts` 规则，阳卦纳阳干、阴卦纳阴干，阳卦地支顺行、阴卦地支逆行，为内外卦各爻配干支
6. **六亲**：使用 `liuqin.ts` 以卦宫五行为"我"，通过五行生克定六亲（生我父母、我生子孙、克我官鬼、我克妻财、同我兄弟）
7. **六神**：使用 `liushou.ts` 按日柱天干定初爻六神起例，每爻顺排一位
8. **空亡**：使用 `kongwang.ts` 根据日柱确定所属旬，获取旬空地支，标注对应爻位
9. **神煞**：使用 `shensha.ts` 按日柱天干地支匹配九种神煞，逐爻标注
10. **伏神**：使用 `fushen.ts` 以宫卦（纯卦）对应爻位的六亲为伏神
11. **旺衰**：使用 `deities.ts` 的 `getWangShuai` 函数，以月建地支判断各爻五行旺相休囚死
12. **三合局**：使用 `sanhe.ts` 检测卦中地支是否构成申子辰、亥卯未、寅午戌、巳酉丑三合局
13. **反吟伏吟**：使用 `fanyin.ts` 对比本卦与变卦的地支，检测反吟（地支相冲）和伏吟（卦象相同）
14. **变卦**：若有动爻，使用 `coin.ts` 的 `getChangedYaos` 将老阳变少阴、老阴变少阳，生成变卦

### 数据层

数据层位于 `src/data/`，以 TypeScript 模块形式提供纳甲筮法的全部静态知识：

- **hexagrams.ts**：六十四卦完整数据库，按八宫（乾、兑、离、震、巽、坎、艮、坤）排列，每宫 8 卦（纯卦 + 一世至归魂）。每卦包含卦名、二进制编码、上下卦、宫位、宫位置编号、卦辞、六爻爻辞。提供按编码、按卦名、按宫位三种查找方式
- **naja.ts**：纳甲规则表。天干规则：乾纳甲壬、坤纳乙癸，其余六卦各纳一干。地支规则：阳卦顺行（子寅辰午申戌），阴卦逆行（丑亥酉未巳卯），内卦从初爻起、外卦从四爻起
- **bazi.ts**：八字计算相关基础数据，涵盖天干地支列表、五行生克完整关系（生我、我生、克我、我克）、地支六合六冲映射、四组三合局定义、六组旬空映射、五虎遁起月干规则、五鼠遁起时干规则
- **deities.ts**：六神起例（按日干定初爻）、九种神煞的详细地支匹配规则、旺相休囚死计算逻辑（当令者旺、我生者相、生我者休、克我者囚、我克者死）
- **jieqi.ts**：二十四节气近似日期表，提供给定日期所在节气区间查询功能

### 状态管理

Pinia Store（`stores/paipan.ts`）管理三个核心状态：

- **currentResult**：当前排盘结果，用于结果页面展示
- **history**：历史记录列表，通过 localStorage 持久化，上限 100 条。加载时自动反序列化日期字符串
- **displayOptions**：七个布尔值配置开关，控制排盘表中各项信息的显隐。支持局部更新

### 组件层

UI 组件位于 `src/components/`，按功能划分：

- **起卦组件**：CoinToss（掷硬币模拟器，含动画反馈、进度条、逐爻结果展示）、ManualInput（六爻选择器，支持上一步回退）
- **展示组件**：HexagramView（卦象卡片，含卦名、卦辞、世应标记、动爻标记、纳甲干支）、PaipanTable（排盘详情表格，列包含爻位、爻象、六神、六亲、伏神、干支、五行、旺衰、神煞、空亡，按显示开关控制列显隐）
- **信息组件**：DateTimeInfo（起卦时间、节气、四柱八字、月建日建、空亡）、InfoToggles（显示开关面板）
- **历史组件**：HistoryList（历史记录列表，显示卦名、时间、八字，支持查看与删除）

### 导出功能

composable 函数 `useExport.ts` 提供：

- **encodeResultToUrl**：将排盘结果序列化为 JSON 并编码为 URL 参数
- **decodeResultFromUrl**：从 URL 参数解码排盘结果
- **exportAsText**：导出为格式化的纯文本文件（含卦名、八字、排盘表、三合局、反吟伏吟等）
- **exportAsJson**：导出为结构化的 JSON 文件

## 使用指南

### 起卦

1. 打开应用进入首页，有三种起卦方式可选
2. **手动起卦（掷硬币）**：点击"掷币"按钮模拟投掷三枚硬币，共投掷六次，从初爻到上爻逐爻生成。每次投掷后显示三正/二正一反/一正二反/三反的结果及对应的爻类型（少阳/少阴/老阳/老阴），动画反馈后进入下一爻
3. **手工指定**：逐爻从少阳、少阴、老阳、老阴四种爻象中选择，支持上一步回退修改，全部选定后确认提交
4. **自动起卦**：系统一次随机生成六个爻，自动进入结果页面，结果同时保存至历史记录

### 解读排盘

结果页面展示：

- **顶部信息区**：起卦时间、当前节气区间、四柱八字、月建日建、空亡信息
- **卦象对比区**：本卦与变卦并排展示，每卦以传统卦画样式呈现（阳爻一横、阴爻两短横），标注爻位、世应、纳甲干支、卦名、卦辞、动爻标记
- **附加信息区**：三合局（含名称、五行属性、参与爻位）、反吟伏吟（含类型与文字描述）
- **排盘详情表**：按爻位逐行展示六神、六亲、伏神（含地支）、干支、五行、旺衰、神煞标签、空亡标记
- **信息开关**：可通过复选框灵活控制卦辞、三合局、反吟伏吟、伏神、旺衰、神煞、空亡等信息的显隐
- **操作按钮**：复制分享链接、导出文本、导出 JSON

### 历史管理

- 每次起卦结果自动保存至浏览器本地存储
- 历史页面按时间倒序排列，显示卦名（及变卦名）、起卦时间、八字
- 支持单条删除和清空全部记录
- 点击历史条目可重新查看完整排盘

### 导出与分享

- **导出文本**：生成可读的纯文本格式排盘报告，包含完整的排盘信息
- **导出 JSON**：导出结构化的 JSON 数据，便于二次处理或存档
- **复制分享链接**：将当前完整排盘编码为 URL 参数，生成链接后可通过即时通讯工具分享给他人

## 开发指南

### 构建验证

```bash
npm run build
```

构建前自动执行 TypeScript 类型检查（vue-tsc），确保类型安全。

### 项目规范

- 类型定义统一存放于 `src/types/index.ts`，避免分散
- 静态数据与业务逻辑分离：`data/` 仅存放知识数据，`engine/` 存放计算逻辑，数据与计算不耦合
- 组件使用 Vue 3 `<script setup>` 语法，组合式 API
- 数据流单向：组件通过 Pinia Store 读写状态，引擎函数为纯函数（输入输出由 TypeScript 类型保障），不直接操作 DOM 或状态
- 颜色主题：传统风格配色，主色深红（#8b0000），背景米白（#f5f0e8），装饰金棕（#8b7355），阳爻红色、阴爻蓝色
